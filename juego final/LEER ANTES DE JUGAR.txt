
Instalacion de python
comenzamos desargando python desde la pagina oficial python.org
buscamos el ejecutable de 32 o 64 bits 
luego procedemos a instalarlo
para saber si funciona abrimos una ventana de comando, y escribimos python
por la cual nos montrara en pantalla Python y la version instalada
